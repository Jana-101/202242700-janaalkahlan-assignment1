## Project Overview
This project is a personal portfolio website built using HTML, CSS, and JavaScript.  
It is designed to be simple, responsive, and easy to navigate across different devices.

---

## File Structure
The project is organized as follows:
- `index.html` contains the main structure and content of the website.
- `css/styles.css` handles the layout, styling, and responsive design.
- `js/script.js` contains JavaScript code for interactivity.
- `assets/images/` stores images used in the project.
- `docs/` contains documentation files, including AI usage and technical documentation.

---

## HTML Structure
The HTML file uses semantic elements such as `header`, `section`, and `footer` to organize content.  
The main sections include About, Projects, Skills, and Contact.  
A form is included in the Contact section for user input.

---

## CSS Styling and Responsiveness
CSS is used to style the layout, colors, fonts, and spacing of the website.  
Responsive design techniques such as Flexbox and media queries are applied to ensure the site works well on desktop, tablet, and mobile devices.

---

## JavaScript Functionality
JavaScript is used to add interactivity to the website.  
A hamburger menu is implemented to toggle the navigation menu on smaller screens, improving usability on mobile devices.
